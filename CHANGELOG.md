# Change Log

## 0.4.0

- Rewrite architecture
- Pure TypeScript compilation (removed webpack)
- Large image support with chunked Transferable ArrayBuffer transfer
- File change watching: auto-refresh on edit, auto-close on delete
- Info bar showing resolution, channels, colorspace, file size, and timing
- Export as PNG via right-click context menu in the editor
- Export as PNG via Explorer right-click context menu
- Status bar integration: image size and zoom level
- Checkerboard background for transparency visualization with theme-aware colors

## 0.3.1

- Move save PNG action to menu

## 0.3.0

- Add save as PNG feature

## 0.1.0

- Easier and faster viewing of files, based on Vscode built in [Image Preview extension](https://github.com/microsoft/vscode/tree/10dc1a34b5/extensions/media-preview)

## 0.0.1

- Initial beta release
